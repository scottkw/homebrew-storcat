# homebrew-storcat

Homebrew tap for [StorCat](https://github.com/scottkw/storcat) — a cross-platform directory catalog manager built with Go and Wails.

## Installation

```bash
brew tap scottkw/storcat
brew install --cask storcat
```

## About this tap

> **This tap is auto-managed.** The cask is automatically updated when a new StorCat release is published. Do not edit `Casks/storcat.rb` manually — changes will be overwritten on the next release.

The source of truth for packaging configuration is:
**[scottkw/storcat/packaging/homebrew/](https://github.com/scottkw/storcat/tree/main/packaging/homebrew)**

## Note

The cask may reference an older version if a release is in progress. Run `brew update && brew upgrade storcat` to get the latest available version.
